# Ghost Digital Clock

マウスオーバーで表示される最小限デジタル時計です。  
背景色、文字色、フォント、サイズ、24/12時間表示、日付表示の有無や位置をカスタマイズできます。

## 機能

- 24時間/12時間表示切替
- 秒表示のオン/オフ
- 日付表示のオン/オフ
- 日付の位置を上または下に変更可能
- 背景色・文字色の変更
- フォントとサイズの変更
- マウスオーバーで時計を表示/非表示
- 全画面切替対応

## 使い方

1. リポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開く
3. 設定パネルから各種カスタマイズが可能
